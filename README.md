들여쓰기의 경우 잘못하면 내가 원하는 값이 아닌 다른 값이 나올때도 있었고 값 자체가 출력이 되지 않을때도 있었다.

:콜론의 경우 for문과 if문의 조건뒤에 항상 붙는것을 인지하고 코딩을 해야할 것 같다. 콜론을 넣어야 할 곳에 넣지 않는다면 값 자체가 
출력이 되지 않았다.
